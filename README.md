## Overview

**TaniSmart** is a modern, fully responsive agricultural technology landing page. It presents an agri-tech brand's mission — empowering farmers with cutting-edge tools, sustainable practices, and data-driven solutions. Built entirely with **HTML5**, **CSS3**, and **Vanilla JavaScript**, the site delivers a clean, professional UI without any framework dependency.

## Features

- **Hero Section** — Full-screen banner with CTA buttons and smooth intro animations
- **Stats Counter** — Animated display of key metrics (100% satisfaction, 20+ years, 160K+ farmers)
- **Agriculture Solutions** — Feature cards covering organic fertilizer, smart irrigation, crop monitoring & soil stewardship
- **Blog / News Section** — Latest agri-tech news with card layout and date tags
- **Testimonials** — Multi-column customer reviews with avatar support
- **CTA Banner** — High-impact call-to-action with dark overlay on hero image
- **Multi-column Footer** — Organized links across Product, Company, Services, Innovations, and Success Stories
- **Fully Responsive** — Optimized for mobile, tablet, and desktop screens
- **No Framework Dependency** — Pure HTML/CSS/JS for maximum performance and simplicity

## Getting Started

No build tools or dependencies required. Just clone and open.

### 1. Clone the Repository

```bash
git clone https://github.com/NSniha/agricultural-landing-page.git
```

### 2. Open in Browser

```bash
cd agricultural-landing-page
# Open index.html directly in your browser
# OR use VS Code Live Server extension
```

That's it — no `npm install`, no build step, no configuration needed.
